Lip Reading Using Neural Networks and Deep Learning

N.Durga Sri, R.Akhil, S. Venkat Durga Prasad, V.Jayanth

Abstract:

Lip reading is a method for understanding words or speech without the use of audio through visual interpretation of face, mouth, and lip movement. This research validates the usage of machine learning, specifically using neural networks, to create a system for automatically reading lips. Two different CNN architectures were trained, and a web application for real-time word prediction using the top-performing model was developed.

Implementation:

The implementation involves using a Convolutional Neural Network (CNN) for lip reading. The methodology includes data gathering, pre-processing, developing a neural network architecture, training, and evaluation. Key components of a CNN model, such as convolutional layers, activation functions, pooling layers, and fully connected layers, are utilized for feature extraction and prediction.

Introduction:

Even experienced lip readers have found lip reading troublesome. This research aims to solve lip reading challenges using machine learning techniques, providing potential benefits for speech recognition in noisy environments and aiding hearing aid technologies. The research explores the use of convolutional neural networks (CNN) for lip reading, addressing the diversity of languages and speech articulations.

Conclusion:

The lightweight model architecture performed better in real-time lip reading. Model D, built on this architecture, had the highest accuracy and performance. The research concludes by acknowledging the mentor and friends who contributed to the project.
